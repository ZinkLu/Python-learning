### Linux上的Shebang 符号(#!)
- `#!`符号叫做`Shebang`或`Sha-bang`
- `Shebang`通常在`Unix`系统脚本中**第一行**开头使用
- 指明**执行这个脚本文件的解释程序**

#### Shebang的使用
1. 用`which`查询`python3`解释器所在的路径
2. 修改要运行的 **主python文件**，在第一行增加内容
3. 修改**主python文件的文件权限**，增加**执行权限**
4. 直接执行
